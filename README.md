Simple boilerplate for LOVE2D, including small script in lua. 

## Installation

Install Love2D
```bash
$ # to come
```
Install the boilerplate
```bash
$ git clone git@github.com:MikaelJG/boiled_love.git
```
Move in the project's root
```bash
$ cd boiled_love/ 
```
Start the game.
```bash
love .
```
